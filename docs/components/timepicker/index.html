#Create Properties

NameTypeDefault ValueDescription
labelstringn/aThe text label that appears with the TimePicker.
labelSidestringtopSide enum specifying whether the label appears on the side or top. Only top and left currently supported.
defaultTimestring(current Time)The default time to set for the TimePicker.

#LabelSide values:

  • bottom
  • left
  • right
  • top

#Element Properties

NameTypeDescription
colorstringSets the TimePicker component's color, doesn't affect label.
timenumberThe Time to set, will ignore values that aren't in the TimeFormat.

*The default time format is: hh:mm:ss
