### Create Properties

| Name   | Type   | Default Value | LRT Name | LRT type |
| ------ | ------ | :-----------: | -------- | -------- |
| text   | string |      n/a      | Text     | string   |
| width  | number |      n/a      |          | number   |
| height | number |      n/a      |          | number   |
